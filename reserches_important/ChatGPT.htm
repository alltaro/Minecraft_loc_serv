json

}

Pour ajouter un nouveau nom, par exemple "David", à la liste des noms, vous pouvez utiliser la commande suivante dans le terminal Linux :

bash
echo '{"noms": ["Alice", "Bob", "Charlie", "David"]}' | jq '.' > data.json

Cette commande utilise jq pour ajouter "David" à la liste existante. Assurez-vous d'avoir jq installé sur votre système. Vous pouvez l'installer en utilisant le gestionnaire de paquets de votre distribution Linux.

Si vous avez déjà un fichier JSON existant et que vous souhaitez l'éditer directement, vous pouvez utiliser jq de la manière suivante :

bash
jq '.noms += ["David"]' data.json > temp.json && mv temp.json data.json

Cette commande utilise jq pour ajouter "David" à la liste existante dans le fichier data.json. Elle crée d'abord un fichier temporaire (temp.json) et le renomme ensuite pour remplacer le fichier d'origine.
